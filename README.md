# 📝 Notes App (Next.js)

A simple full-stack Notes App built using Next.js.
You can add notes, delete them, and generate summaries for each note.

---

## 🚀 Features

* ➕ Add new notes
* 🗑️ Delete notes
* ✨ Summarise each note
* 💾 Data stored in localStorage
* 🔗 Frontend + Backend connected

---

## 🛠️ Tech Stack

* Next.js
* React (useState, useEffect)
* CSS
* API Routes (Backend)

---

## 📂 Project Structure

* `/app` → Main frontend pages
* `/api/summarise` → Backend API route
* `notes.css` → Styling

---

## ⚙️ How It Works

1. User writes a note
2. Clicks "Add Note" or presses Enter
3. Notes are saved in localStorage
4. Click "Summarise" → text goes to backend
5. Backend returns a short summary
6. Summary is displayed under the note
